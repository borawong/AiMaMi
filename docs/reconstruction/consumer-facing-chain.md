# Consumer Facing Chain

The public reconstruction chain should start from sanitized root status/spec files, then follow audit maps into logic/frontend summaries and distilled 1.0.9 leaves. Non-public execution material is excluded from this repository copy.

## Public Inputs

- Gate and output specs: `evidence/full-chain/internal/root/`
- Audit summaries: `evidence/full-chain/internal/audits/`
- Distilled leaves and ledgers: `evidence/full-chain/internal/leaves/`
- Frontend contract/control-flow/IPC/source-map files: `evidence/full-chain/internal/frontend-map/`

## Excluded Classes

Binaries, full logs, sessions, cache/build/dump/assets output, local machine roots, execution queues, agent instruction documents, tool engineering trees, and outtake material are not imported.
