# OpenAiMami

OpenAiMami 是一个面向个人本地工作流的桌面应用。本仓库公开已经脱敏的可审计材料和当前公开源码，目的不是发布商业介绍，也不是保留闭源黑箱，而是让项目能够在公开许可下继续被检查、复核和迭代。

## 为什么开源

1. 个人迭代：公开材料和源码可以让 OpenAiMami 的后续改进不再依赖私有上下文，任何变更都应能回到仓库内证据、实现和文档中复核。
2. Apache License 许可：项目沿用 [Apache License](LICENSE) 发布，允许使用、修改、分发和继续补全，同时保留清楚的许可边界。
3. 用得更放心：公开实现链路、重建材料和脱敏边界，是为了让使用者能检查本地数据、缓存、命令、导入导出和系统集成如何被处理，降低隐私泄露和不透明执行的风险。

如果有人基于仓库内公开材料还原出更完整的 OpenAiMami 实现，欢迎以 PR 形式提交。PR 应说明使用了哪些仓库内证据、做了哪些校验、哪些行为已经还原、哪些仍是骨架或待实现项。
